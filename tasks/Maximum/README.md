# Maximum Element in an Array

This program reads an array of integers and finds the maximum value.

# Problem

Given an array of n integers, find the largest element in the array.

# Input

The first line contains an integer n — the size of the array.

The second line contains n integers x₁, x₂, …, xₙ.

# Output

Print one integer — the maximum value in the array.

# Constraints

1 ≤ n ≤ 2 · 10⁵  
-10⁹ ≤ xᵢ ≤ 10⁹

# Approach

The program iterates through the array while keeping track of the maximum value seen so far.

# Algorithm

- Read n  
- Read array elements  
- Initialize maximum value  
- Update maximum while iterating  
- Print result

# Complexity

Time complexity: O(n)  
Space complexity: O(n)
