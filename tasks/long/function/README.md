# Geometry Calculator (C++)

## Description
This is a simple console-based C++ program that calculates different geometric values based on user input. The program uses a menu system and function overloading to compute areas and circumference for basic shapes.

---

## Features
- Area of Circle
- Area of Rectangle
- Area of Square
- Area of Triangle
- Circumference of Circle
- Input validation (only positive numbers allowed)
- Reusable input function (`getPositiveN`)
- Function overloading for `calculateArea`

---

## How it works
The program shows a menu:

1. Circle  
2. Rectangle  
3. Square  
4. Triangle  
5. Circumference  
0. Exit  

User selects an option and enters required values. The program then calculates and displays the result.

---

## Key Concepts Used
- Functions
- Function Overloading
- Loops (`while`, `do-while`)
- `switch` statement
- Input validation
- Modular code design

---

## Notes
- All inputs must be positive numbers.
- The program uses `3.14` as an approximation of π.
